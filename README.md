# Letter MORF

A unique daily word puzzle game where letters transform after every word you create!

**Version 6.6 - Production Ready**

## How to Play

Transform letters to create as many 5-letter words as you can before running out of moves!

- **Start with 20 moves**
- Change any letter (costs 1 move per change)
- Submit valid 5-letter words to earn moves back
- After each word, letters automatically **MORF**:
  - Vowels transform to different vowels
  - Consonants shift to the next consonant
- Game ends when you run out of moves or click "End Game"
- Remaining moves = bonus score!

## Features

- **Daily Puzzles** - New letter combination every day
- **Smart Scoring** - Letter frequency + word rarity bonuses
- **Mobile Friendly** - Touch controls & swipe gestures
- **Progress Tracking** - 7-day rolling average
- **Practice Mode** - Unlocks after completing all 7 days

## Scoring

### Letter Values:
- Common (A, E, I, O, T, N, S, H, R, D): 1 pt
- Frequent (U, L, C, M, W, F): 2 pts
- Uncommon (G, Y, P, B, V): 3 pts
- Rare (K, J, X, Q, Z): 4 pts

### Word Frequency Bonus:
- Common: +1 pt
- Frequent: +2 pts
- Regular: +3 pts
- Uncommon: +4 pts
- Rare: +5 pts

**Final Score = Earned Moves × 10**
